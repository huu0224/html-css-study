Fintech Data Visualization

핀테크 실시간 자산 대시보드 최적화

수만 개의 금융 데이터를 실시간으로 수집하고,
브라우저 과부하 없이 시각화하여 투자 판단을 돕는 관리자 시스템 구축기

Background & Problem

초당 수천 건씩 갱신되는 암호화폐 및 주식 데이터를 차트에 반영하는 과정에서 Main Thread 점유율이 90%를 상회하는 문제가 발생했습니다. 이로 인해 사용자가 버튼을 클릭하거나 메뉴를 이동할 때 1~2초간 화면이 멈추는(Jank) 현상이 빈번해져 서비스 이탈률이 증가했습니다.

Technical Solution

Web Worker 도입

데이터 파싱, 필터링, 기술 지표 계산 등 복잡한 연산 로직을 별도의 백그라운드 스레드로 분리하여 UI 응답성을 확보했습니다.

Canvas 기반 렌더링

수만 개의 DOM 요소를 생성하는 SVG 대신, 고성능 픽셀 렌더링이 가능한 HTML5 Canvas를 사용하여 GPU 가속을 활용했습니다.

또한, React Query의 staleTime을 정교하게 조정하여 불필요한 네트워크 요청을 줄이고, Virtual Scroll을 적용하여 수천 개의 데이터 리스트 중 사용자 화면에 보이는 영역만 렌더링하도록 최적화했습니다.

Key Outcome
  • 데이터 로딩 및 연산 속도 70% 단축 (3.2s → 0.9s)
  • 실시간 업데이트 시 브라우저 프레임 드랍 제거 (60fps 고정)
  • 복잡한 멀티 차트 환경에서도 메모리 사용량 40% 절감
  • 사내 트레이더 업무 효율성 지표 25% 향상
